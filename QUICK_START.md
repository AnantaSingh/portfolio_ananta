# Quick Start Guide

## Running Locally

To see your changes locally, run:

```bash
npm run dev
```

This will start the development server. Open your browser and navigate to:

**http://localhost:5173**

The page will automatically reload when you make changes to the code.

## Available Routes

- `/` - Home page (Hi, I am Ananta)
- `/about` - About page with work experience and skills
- `/projects` - Projects showcase
- `/resume` - Resume PDF viewer with download button
- `/contact` - Contact information

## Making Changes

1. Edit files in the `src/` directory
2. The browser will automatically refresh with your changes
3. No need to restart the server

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `dist/` folder.

## Preview Production Build

```bash
npm run preview
```

This serves the production build locally at `http://localhost:4173` so you can test how it will look when deployed.
